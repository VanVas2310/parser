# Парсинг новостного сайта

Парсинг новостной страницы "Известия" с группировкой новостей по разделам.

## О проекте
Проект для парсинга новостной страницы сайта "Известия" с группировкой новостей по тематическим разделам. Программа извлекает заголовки и ссылки на новости, сохраняя результаты в структурированном JSON-формате.

**Источник данных:** Веб-архив страницы https://web.archive.org/web/20230903112115/https://iz.ru/news

## Функциональность
- Парсинг HTML-страницы новостного сайта
- Группировка новостей по разделам (Общество, Здоровье, Спорт и т.д.)
- Извлечение заголовков и ссылок на новости
- Сохранение результатов в JSON-формате
- Каждый раздел содержит минимум одну новость

## Требования
- Python 3.6 или выше
- Библиотеки: requests, beautifulsoup4, lxml
